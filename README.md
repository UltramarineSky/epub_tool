# 怎么使用？
## python执行
1. 下载python3.8
2. 使用`git clone https://github.com/cnwxi/epub_tool.git`克隆本仓库，或直接在网页下载源码压缩包，解压后得到py文件
3. 使用命令行执行 `python path_of_your_py_files/**.py`
## exe
1. 从[releases](https://github.com/cnwxi/epub_tool/releases)下载对应的exe文件
2. 双击运行
